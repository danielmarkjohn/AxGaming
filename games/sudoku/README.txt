TriLevel Sudoku
================

Files:
- index.html
- style.css
- script.js

Features:
- Landing page + Menu + Game + Victory screens.
- 3 difficulty levels (easy, medium, hard) with different number of removals.
- Random fresh board every New Game; generator produces a full solution then removes cells while attempting uniqueness.
- Notes (pencil marks), on-screen keypad, keyboard support, hints, validate, erase.
- Timer and best-times saved per difficulty in localStorage.
- Nice victory confetti visualization.
- Fully responsive and touch-friendly.

How to run:
1. Unzip the archive.
2. Open index.html in a modern browser.
